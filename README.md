#Блог на PHP з використанням BOOTSTRAP#

Це динамічний сайт, який працює на php, та фронт розроблений за допомогою BOOTSTRAP.
Я обрав Bootstrap так як в ньому є мінімальні адаптивні принципи, він містить готові компоненти та стилі, такі як кнопки, форми, таблиці.
Але всеодно Адаптив треба дороблювати.
